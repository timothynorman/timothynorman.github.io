Tripped Up With Data Types

 2023-06-29  2 minutes  Coding practice Lesson learned 
I’ve taken some time this week to get back into solving coding practice problems during some downtime at school. One issue I’ve had with this semester is that there isn’t as much coding built into the course work, so I really have to take it upon myself to find things to code. I was working though Project Euler question 14 where you perform iterative calculations on values up to one million. I worked out the logic, ran my code and found that the longest chain in the Collatz Sequence started at value 910107… only for it to be wrong. I’ve been tripped up before with Project Euler questions where the values are so large that they overflow the capacity of an int to hold them, but since I was working with values only up a million I figured it would be fine.
