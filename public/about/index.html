About Me

 2023-06-25  1 minute  No tag

Hi, and welcome to my blog / portfolio (blogfolio?). My name is Tim and I’m a computer programming student at Algonquin College in Ottawa, Canada. I’m primarily studying Object Oriented Programming using Java, relational databases, and some web development sprinkled in.

This site is my attempt to document my “Lessons Learned” so that I can refer back to them if I ever need to, but also help out anyone else who stumbles across it. Considering how many tutorials, help posts, and walkthroughs I’ve gone through it’s only fair that I try to put some more of that good karma back out there.

I will also feature some projects and personal work that I think might be interesting. Right now I’m focused on improving my programmer thinking by working though problem sets (Advent of Code, Project Euler, and LeetCode (following the Neetcode Roadmap), but I want to start tackling larger personal projects as well.
